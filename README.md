👉 https://cp-point.github.io/portfolio

---

## About This Project

이 포트폴리오는 제가 참여한 프로젝트와  
개발 경험을 정리한 사이트입니다.

단순 결과물이 아닌,
**어떻게 설계했고 어떤 문제를 해결했는지**에 집중했습니다.

---

## What I Focus On

- UI를 구현하기 전에 **구조와 흐름을 먼저 설계**합니다.
- 재사용 가능한 **컴포넌트 중심 아키텍처**를 지향합니다.
- 인터랙션은 시각적 효과가 아니라 **사용자 경험 강화 목적**으로 사용합니다.
- 유지보수를 고려한 파일 구조와 데이터 구조를 설계합니다.

## Key Features

- 프로젝트 상세 페이지 **동적 라우팅 구현**
- 프로젝트 구조에 맞춘 `main + subImages` 데이터 설계
- 공통 레이아웃 기반 컴포넌트 분리
- 프로젝트 특성상 실제 화면 공개가 어려운 경우,  
  구조 기반 비주얼로 대체 설계

---

## 📂 Project Structure

```
src/
 ├── components/     // 재사용 UI 컴포넌트
 ├── pages/          // 라우팅 단위 페이지
 ├── data/           // 프로젝트 데이터 관리
 ├── assets/         // 이미지 및 정적 리소스
 └── styles/         // SCSS 구조 관리
```

> 유지보수와 확장성을 고려하여 기능 단위로 구조화했습니다.
